## 📦安装
```bash
npm install lemons-piano
```

## 🔨用法
``` tsx
//引入组件
import Piano from 'lemons-piano'
//引入组件所使用的样式
import 'lemons-piano/dist/style.css'

function App() {
  
  return (
    <div className="App">
      <!--直接使用组件即可-->
      <Piano></Piano>
    </div>
  )
}

export default App

```
